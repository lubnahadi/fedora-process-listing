# fedora-process-listing
## Overview
This project demonstrates how to view running processes in linux using Fedora.

## Step 1 - List all running processes
Command used:
'ps -ef'
This command shows all active processeswith details like user, PID and command.
![Step 1](fedora.jpg)

## Step 2 - View processes with scrolling
Command used:
'ps -ef | less'
This command allows scrolling through long lists of processes.
![Step 2](fedora .jpg2.jpg)

## Step 3  - Exit from process view
Command used:
'q'
This command exits the scroll view.
![Step 3](fedora.jpg3.jpg)
